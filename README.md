## Getting Started

This project uses react as frontend, laravel as backend & MySql As DB.

### Installation
1. Go to be-laravel directory
2. run 
   ```sh
   composer install
   php artisan migrate
   php artisan storage:link
   php artisan serve
   ```
3. Import user.sql and users.sql
4. Go to fe-react directory
   ```sh
   npm install
   npm run dev
   ```
